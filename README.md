# Weather Dashboard

A modern weather application built with React that provides real-time weather information and a multi-day forecast for cities around the world. The application integrates with the OpenWeather API to deliver accurate weather data while providing a clean, responsive, and user-friendly interface.


## Overview
Weather Dashboard allows users to search for any city and instantly view:

- Current weather conditions
- Temperature and weather descriptions
- Weather-specific icons
- Multi-day forecast
- Loading and error states for a smooth user experience

The project was built to strengthen frontend development skills in React, asynchronous data fetching, state management, API integration, conditional rendering, and component-based architecture.

## Features

### Current Weather Search

Users can search for any city and retrieve current weather information including:

- City name
- Temperature
- Weather condition
- Humidity
- Wind speed
- Atmospheric information

---

### Multi-Day Forecast

The application displays forecast data for upcoming days by processing OpenWeather forecast responses and extracting representative daily weather information.

Displayed forecast data includes:

- Day label (Today, Tomorrow, Mon, Tue, etc.)
- Weather icon
- Weather description
- Maximum temperature
- Minimum temperature

---

### Error Handling

The application gracefully handles:

- Invalid city names
- Empty search input
- API request failures
- Network errors
- Invalid API keys

Users receive meaningful feedback instead of application crashes.

---

### Loading States

A loading indicator is displayed while weather information is being fetched, improving user experience and providing visual feedback during asynchronous operations.

---

### Responsive Design

The application is designed to work across:

- Mobile devices
- Tablets
- Laptops
- Desktop screens
